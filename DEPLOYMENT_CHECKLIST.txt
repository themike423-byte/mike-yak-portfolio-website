TROUBLESHOOTING:
----------------

ISSUE: "Git push fails with authentication error"
FIX: 
  - Verify you're logged into GitHub
  - Check if you need to set up SSH keys
  - Try: git config credential.helper store

ISSUE: "PowerShell script won't run"
FIX:
  - Run: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  - OR use manual git commands instead

ISSUE: "Changes not showing on live site"
FIX:
  - Wait 2-3 minutes for Vercel build
  - Hard refresh browser (Ctrl+Shift+R or Cmd+Shift+R)
  - Check Vercel dashboard for build status

ISSUE: "Some images/logos not loading"
FIX:
  - logo.clearbit.com may be rate-limited
  - Wait a few minutes and refresh
  - Check browser console for specific errors

ISSUE: "Expandable sections don't work"
FIX:
  - Check JavaScript console for errors
  - Verify togglePartnerCategory function is present
  - Try different browser
